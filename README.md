# DotNames-Js : A Javascript Library To Resolve Names On All Blockchains

## Overview of the API

### Installation

Install @dotlabs/dotnames-sdk

```
yarn add @dotlabs/dotnames-sdk
```


### Getting Started

Setup 
```
  const dotnames = new DotNamesSDK();
```

### exports

```
default - DotNamesSDK
```
